Политики

Правила использования данных

  1. Не используйте данные Wild Boar для имитации наших продуктов или бренда.
  2. При публикации индексов указывайте источник и ссылку на wildboar.finance.
  3. Коммерческое использование требует соглашения и соблюдения кредитования авторства.

Мы оставляем за собой право приостановить доступ к API в случае нарушения правил или угрозы инфраструктуре.

Индексы

DeFi Index

Исторические дневные OHLC данные по индексу децентрализованных финансов.

Индекс DeFi фиксирует динамику ключевых протоколов децентрализованных финансов. Данные нормализуются, приводятся к UTC и проходят многоуровневую валидацию.

GET

V1

https://wildboar.space:8001/api/defi_index

Возвращает дневные значения индекса DeFi за выбранный период. По умолчанию предоставляет весь доступный диапазон.

Доступ

Public

Rate limit

30 req/min

Кэширование

Регенерация данных один раз в сутки

Заголовки

Параметр Обязательный Описание
Access-Control-Allow-Origin Нет *, response header (CORS).
Access-Control-Allow-Methods Нет GET, response header (CORS).
Access-Control-Allow-Headers Нет *, response header (CORS).

Структура ответа

Поле Тип Описание
time string (date) Дата торгового дня в формате YYYY-MM-DD.
open number Значение индекса на открытии сессии.
high number Максимальное значение индекса за день.
low number Минимальное значение индекса за день.
close number Значение индекса на закрытии сессии.

Пример ответа

[
 {
  "time": "2020-08-28",
  "open": 1012.0,
  "high": 1189.0,
  "low": 964.1,
  "close": 1018.8
 },
 {
  "time": "2020-08-29",
  "open": 1017.5,
  "high": 1096.5,
  "low": 1015.0,
  "close": 1056.0
 }
]

  1. Данные представляют агрегированный индикатор DeFi сегмента и не являются торговой рекомендацией.
  2. Ряды могут корректироваться в случае обнаружения ошибочных котировок или делистинга источников.
  3. Поля возвращаются в формате с плавающей точкой, точность до десятых.
